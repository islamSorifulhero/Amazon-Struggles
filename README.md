## ✨ Features

- **Fully Responsive** — works on mobile, tablet, and desktop
- **Scroll Animations** — fade-up reveal on scroll using IntersectionObserver
- **Interactive Calendar** — month navigation, date selection, weekends disabled
- **Tab Switcher** — Mission / Vision toggle in Approach section
- **Animated Atom Icon** — CSS keyframe rotation in Why Choose section
- **Sticky Navbar** — stays fixed on scroll
- **Smooth Scroll** — all CTA buttons scroll to booking section


## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|------------|--------|
| `> 900px` | Full desktop layout |
| `≤ 900px` | Tablet — stacked sections, 2-col grids |
| `≤ 640px` | Mobile — single column, hidden nav links |
